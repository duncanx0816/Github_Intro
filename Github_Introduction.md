### 如何在使用git来添加标记并上传到github

```
git tag "v1.1"                 #为当前提交添加标记
git add .                      #添加修改当暂存区
git commit -m "添加标记"        #提交修改
git push --tags origin master  #推送提交到远程仓库。并且顺带推送标记
```
